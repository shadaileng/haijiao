```javascript
// 将 [emoji]code[/emoji] 替换为 <img> 
function m(e) {
    var t = d(e);  // 提取所有 [emoji]code[/emoji]
    return 0 != t.length && t.forEach(t => {
        e = e.replace(`[emoji]${t}[/emoji]`, 
            `<img src="${c.a + l[a].img}" data-emoji="${t}"/>`)
    }), e
}

// Emoji 图片路径前缀
// "0a46": function(e, t, a) {
//     t.a = "/images/emoji/"

### Emoji 资源

参考代码中 emoji 映射表（模块 `b1fb`）包含数百个 emoji 条目，格式如：

```
"0": { "img": "0.gif", "showName": "😀" }

### 方案

#### 功能要点

- 实现 `renderEmoji(text)` 纯函数
- 从参考代码中提取 emoji 映射表，或直接使用正则替换
- 可选择预下载 emoji 映射表为 JSON，或仅在渲染时按需替换

## 依赖关系

```
1. 每日签到       — 无依赖
2. 帖子分类展示     — 无依赖
3. 帖子收藏        — 无依赖
4. 关系扩展        — 无依赖
5. 用户信息展示     — 无依赖
6. 验证码交互      → 依赖于登录功能（已有）
7. 标签浏览        — 无依赖
8. Emoji 解析     — 无依赖
9. 视频播放器      → 需先安装 hls.js + DPlayer
10. VIP 信息展示   — 无依赖
11. 注册/找回密码  → 依赖于验证码交互（6）
```

- 功能 1-5、7-8、10 可并行开发
- 功能 6、9 可独立开发
- 功能 11 需要功能 6 完成后开发
