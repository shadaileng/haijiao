## 1. 功能列表

| 功能 | 入口 | 说明 |
|:----|:----|------|
| 登录 | `/login` | 邮箱/用户名 + 密码登录，或手动配置 UID/Token |
| 帖子详情 | 首页 → 帖子详情 | 输入帖子 ID 查看内容、附件和图片 |
| 用户帖子 | 首页 → 用户帖子 | 输入用户 ID 查看该用户的所有帖子 |
| 搜索 | 首页 → 搜索 | 按关键词搜索帖子 |
| 关注列表 | 首页 → 关注 | 输入 UID 和 Token 查看关注列表 |
| 图片查看 | 帖子详情 → 查看所有图片 | 全屏浏览图片 |
| 设置 | 底部 Tab → 设置 | 配置 UID/Token/API 地址、退出登录 |

## 2. 快速入门

### 2.1 登录

**方式一：自动登录（推荐）**

1. 打开应用 → 自动跳转到登录页
2. 输入**邮箱**或**用户名**和**密码**
3. 点击「登录」
4. 登录成功后自动跳转首页

**方式二：手动配置**

1. 在登录页点击「手动配置 UID/Token」
2. 输入从海角社区获取的 **UID** 和 **Token**
3. 点击「保存」

### 2.2 查看帖子

1. 首页点击「帖子详情」
2. 输入帖子 ID
3. 查看帖子标题、内容、附件
4. 图片附件可点击「查看所有图片」全屏浏览
5. 视频附件点击后在浏览器新标签页播放

### 2.3 搜索帖子

1. 首页点击「搜索」
2. 输入关键词
3. 浏览搜索结果，点击可跳转到帖子详情
4. 支持无限滚动加载更多结果

### 2.4 浏览用户帖子

1. 首页点击「用户帖子」
2. 输入用户 ID
3. 浏览该用户发布的所有帖子（无限滚动）
4. 点击帖子可查看详情

### 2.5 关注列表

1. 首页点击「关注」
2. 输入目标用户的 **UID** 和 **Token**
3. 查看关注列表
4. 点击关注用户可查看其帖子

### 2.6 数据源配置

在设置页可自定义 API 数据源地址，默认使用 `haijiao.com`。

## 3. 设置

| 设置项 | 说明 |
|:-------|------|
| 登录状态 | 显示当前登录用户昵称和 UID |
| 退出登录 | 清除所有登录凭据 |
| UID | 手动配置用户 ID |
| Token | 手动配置认证令牌 |
| API 地址 | 自定义数据源域名（默认 haijiao.com） |
| 数据源 URL | 显示当前 API 代理地址 |

## 4. 常见问题

### Q: 如何获取 UID 和 Token？

登录成功后系统自动获取。如需手动配置，请在海角社区登录后从浏览器开发者工具中提取请求头 `X-User-Id` 和 `X-User-Token`。

### Q: 帖子内容无法显示？

- 检查网络连接
- 确认帖子 ID 正确
- 尝试在设置页切换数据源地址
- 部分帖子可能需要正确的登录凭证

### Q: 视频无法播放？

当前版本视频通过浏览器新标签页打开，需确保：
1. 浏览器允许弹窗
2. 已登录海角社区（视频需要认证）
3. 网络可访问海角社区

### Q: 如何清除本地缓存？

在设置页点击「清除凭证」或退出登录即可清除缓存的用户数据。

### Q: 图片加载失败？

- 图片使用自定义加密传输，部分图片可能需多次尝试
- 可尝试在外置浏览器中打开帖子链接查看原始图片
