## 1. 功能列表

| 功能 | 入口 | 说明 |
|:----|:----|------|
| 登录 | `/login` | 邮箱/用户名 + 密码登录，自动获取认证信息 |
| 热门帖子 | `/hot` | 浏览热门帖子列表，点击进入详情 |
| 帖子详情 | 热门 → 帖子详情 | 查看帖子内容、附件、视频（内联播放）和评论 |
| 搜索 | TabBar → 搜索 | 按关键词搜索帖子 |
| 用户主页 | 点击用户昵称/头像 | 查看用户信息、帖子列表、跳转关注列表 |
| 关注列表 | TabBar → 关注 | 查看关注用户列表 |
| 图片查看 | 帖子详情 → 点击图片 | 全屏浏览图片 |
| 设置 | TabBar → 设置 | 查看登录状态、配置镜像源 |

## 2. 快速入门

### 2.1 登录

1. 打开应用 → 自动跳转到登录页
2. 输入**邮箱**或**用户名**和**密码**
3. 点击「登录」
4. 登录成功后自动跳转热门页

> 登录使用 MD5 签名算法，密码不会明文传输。

### 2.2 浏览热门帖子

1. 热门页展示帖子列表（支持无限滚动）
2. 点击帖子标题或封面进入详情页
3. 帖子详情包含内容、附件和评论

### 2.3 视频播放

1. 在帖子详情页，点击视频封面
2. 视频通过 DPlayer 内联播放（支持 HLS 流媒体）
3. 无需跳转新标签页

### 2.4 搜索帖子

1. 点击底部 TabBar「搜索」
2. 输入关键词回车
3. 浏览搜索结果，点击跳转到帖子详情
4. 支持无限滚动加载更多结果

### 2.5 用户主页

1. 在帖子详情或评论中点击用户昵称/头像
2. 跳转到用户主页，查看用户信息和帖子列表
3. 可查看关注列表

### 2.6 数据源配置

在设置页「镜像源（数据源地址）」可自定义 API 数据源地址。**官方域名 `haijiao.com` 在国内被屏蔽**，需填写后台提供的可用镜像域名。

## 3. 设置

| 设置项 | 说明 |
|:-------|------|
| 登录状态 | 显示当前登录用户昵称和 UID（已登录时） |
| UID | 当前用户 ID（只读） |
| Token | 当前认证令牌（只读） |
| 镜像源（数据源地址） | 自定义数据源域名（默认 haijiao.com，国内不可达） |

## 4. 常见问题

### Q: 帖子内容无法显示？

- 检查网络连接
- 确认帖子 ID 正确
- 尝试在设置页切换数据源地址（填写可用的镜像域名）
- 部分帖子可能需要正确的登录凭证

### Q: 视频无法播放？

1. 确认已登录海角社区（视频需要认证）
2. 确认配置页「数据源」填入了可用的镜像域名
3. 网络需可访问镜像源

### Q: 图片加载失败？

- 图片使用自定义加密传输，部分图片可能需多次尝试
- 可尝试在外置浏览器中打开帖子链接查看原始图片

### Q: 如何清除本地缓存？

在浏览器开发者工具中清除 localStorage 即可。

### Q: `errorCode=1` 是什么意思？

`errorCode=1` 是通用的请求失败码，**不一定是登录过期**。可能原因包括网络问题、参数错误等。如频繁出现，请检查网络连接或重新登录。
