## 1. 通用类型

### 1.1 ApiResult

通用 API 响应包装，所有接口都经过此格式。

```typescript
interface ApiResult<T = any> {
  success: boolean       // 请求是否成功
  data: T                // 响应数据（加密时返回 string）
  isEncrypted?: boolean  // data 是否经过三重 Base64 加密
  errorCode?: number     // 错误码（0=成功）
  message?: string       // 错误消息
}
```

| 字段 | 类型 | 必填 | 说明 |
|:-----|:----|:----:|------|
| `success` | `boolean` | ✅ | `true` 表示成功 |
| `data` | `T` | ✅ | 明文 JSON 或加密字符串 |
| `isEncrypted` | `boolean` | ❌ | `true` 时需执行 `decodeMultiBase64` |
| `errorCode` | `number` | ❌ | 错误码 |
| `message` | `string` | ❌ | 失败时的错误描述 |

**解密流程**：

```
data(string) → atob() × 3 → TextDecoder('utf-8') → JSON.parse → 原始数据
```

### 1.2 toCamelCase 转换

`request()` 函数对所有 API 响应执行 `toCamelCase()` 转换，snake_case 字段自动映射为 camelCase：

| snake_case | camelCase |
|:-----------|:----------|
| `topic_id` | `topicId` |
| `like_count` | `likeCount` |
| `create_time` | `createTime` |
| `comment_count` | `commentCount` |
| `user_id` | `userId` |
| `cover_url` | `coverUrl` |
| `remote_url` | `remoteUrl` |
| `key_path` | `keyPath` |
| `lite_content` | `liteContent` |
| `video_time_length` | `videoTimeLength` |
| `search_type` | `searchType` |
| `total_page` | `totalPage` |
| `reply_id` | `replyId` |
| `commend_list` | `commendList` |

## 2. 用户类型

### 2.1 UserInfo

用户认证信息（存储在 localStorage）。

```typescript
interface UserInfo {
  uid: string           // 用户 ID
  token: string         // 认证令牌
  nickname?: string     // 用户昵称
}
```

| 字段 | 类型 | 说明 | 来源 |
|:-----|:----|------|:----:|
| `uid` | `string` | 用户唯一标识 | 登录响应 `user.id` |
| `token` | `string` | API 认证令牌 | 登录响应 `token` |
| `nickname` | `string` | 用户昵称（可选） | 登录响应 `user.nickname` |

### 2.2 FollowUser

关注列表中的用户。

```typescript
interface FollowUser {
  userId: string     // 用户 ID
  nickname: string   // 用户昵称
}
```

## 3. 帖子类型

### 3.1 TopicItem

```typescript
interface TopicItem {
  topicId: string
  id: string
  title: string
  content: string            // HTML 格式正文
  desc: string
  user: {
    id: string
    nickname: string
  }
  createTime: string         // 创建时间（已转换为 camelCase）
  category?: string          // 分类
  remoteUrl?: string         // 附件远程地址
  coverUrl?: string          // 封面地址
  liteContent?: string       // 精简内容
  videoTimeLength?: number   // 视频时长
  likeCount?: number         // 点赞数
  commentCount?: number      // 评论数
  [key: string]: any         // 扩展字段
}
```

### 3.2 AttachmentItem

```typescript
interface AttachmentItem {
  id: string
  category: string          // 分类: "image" | "video" | ...
  remoteUrl: string         // 远程地址（需要解码）
  coverUrl?: string         // 封面地址
  name?: string
  playInfo?: {
    cdnUrl?: string         // CDN 地址（已转换为 camelCase）
  }
  videoUrl?: string
  url?: string
  [key: string]: any
}
```

### 3.3 CommentItem

```typescript
interface CommentItem {
  userId: number            // 用户 ID
  nickname: string          // 用户昵称
  floor: number             // 楼层
  avatar: string            // 头像
  attachments: string       // 附件
  certVideo: boolean        // 认证视频
  replyId: number           // 回复 ID
  createTime: string        // 创建时间
  content: string           // 评论内容
  commendList: CommentItem[] // 子评论列表
}
```

### 3.4 TopicListResponse / SearchResponse

```typescript
interface TopicListResponse {
  results: TopicItem[]
  page: {
    total: number    // 总数
    page: number     // 当前页
    limit: number    // 每页条数
  }
}

interface SearchResponse {
  results: TopicItem[]
  page: {
    total: number
    page: number
    limit: number
  }
}
```

## 4. 登录类型

### 4.1 LoginParams

```typescript
interface LoginParams {
  username: string         // 邮箱或用户名
  password: string         // 明文密码
  captchaCode?: string     // 验证码（可选）
  captchaId?: string       // 验证码 ID（可选）
  ref?: string             // 来源（可选）
}
```

### 4.2 LoginResponse

```typescript
interface LoginResponse {
  token: string            // 认证令牌
  user: {
    id: number             // 用户 ID
    nickname: string       // 昵称
    avatar: string         // 头像标识
    email: string          // 邮箱
    username: string       // 用户名
    description: string    // 个人描述
    topicCount: number     // 帖子数
    commentCount: number   // 评论数
    fansCount: number      // 粉丝数
    favoriteCount: number  // 收藏数
    createTime: string     // 注册时间
    lastLoginTime: string  // 最后登录时间
    [key: string]: any
  }
  domain: string           // 大陆域名
  domainAbroad: string     // 海外域名
  customerService: string  // 客服邮箱
  ref: string              // 来源
  type: number             // 用户类型
  vipDomain: string        // VIP 域名
}
```

### 4.3 字段映射表

| LoginResponse 字段 | 用途 | 存储位置 |
|:------------------|:-----|:---------|
| `user.id` | UID | `settings store.uid` |
| `token` | 认证令牌 | `settings store.token` |
| `user.nickname` | 界面展示 | `settings store.nickname` |
| `domain` | API 主机 | `settings store.apiBase` |
